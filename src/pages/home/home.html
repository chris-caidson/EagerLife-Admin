EagerLife and EagerLife Admin have both been created as Progressive Web Applications (PWA's) using the Ionic Framework. They use Google's Firebase Cloud Firestore for storing data. Visual Studio Code was used as the code editor for both projects.

GitHub Information

EagerLife and EagerLife Admin have repositories on Github. Both projects use the chris-caidson account.


EagerLife is located at: https://github.com/chris-caidson/EagerLife


EagerLife Admin is located at: https://github.com/chris-caidson/EagerLife-Admin

Ionic Information

EagerLife and EagerLife Admin were both created using Ionic Pro with the chris-caidson account. Both applications use the sidemenu template.


EagerLife is located at: https://dashboard.ionicjs.com/app/a0653596/code/builds/list


EagerLife Admin is located at: https://dashboard.ionicjs.com/app/a8b12395/code/builds/list
